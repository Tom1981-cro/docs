## Login issues

- **Can't sign in?** Reset your password from the login page.
- **Two-factor code not arriving?** Check spam or use a backup code.

## Assessment errors

- **Classification blocked?** Ensure all required fields are completed.
- **Export failing?** Try again in a few minutes, or contact support if the issue persists.

## API errors

- **401 Unauthorized** — check your API key is valid and has the correct scopes.
- **429 Rate limited** — reduce request frequency or upgrade your plan.

## Still stuck?

[Reach out to support](/help-centre/contact-support) with a description of the issue and any error messages.
